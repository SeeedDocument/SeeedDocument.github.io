    Grove IoT Developer Kit Microsoft Azure Edition

    Grove IoT Developer Kit – Microsoft Azure Edition 包含一个 Intel® Edison module， 一个 Intel® Edison for Arduino board，一个 Grove Base Shield, 一组 Grove sensors 和内置 Grove 接口的 actuators 用来快速建模。接上模块即可使用。

    Intel® Edison For Arduino¶

    Intel® Edison compute module 为专业创客和一些工业物联网应用而设计，提供一系列性能优良的商用产品原型开发。它的扩展口与众多的 Arduino 扩展板兼容，Intel® Edison for Arduino 可以通过开源硬件和软件开发环境进行快速方便的原型开发。

    产品特性¶

    • 具有强劲性能的 Intel® Edison 和丰富特性的 Grove 系统的结合

    • 物联网项目的快速原型设计

    • 支持 Arduino、C/C++ 开发，也支持 Node.JS，Python，RTOS，未来会支持可视化编程

    • 提供设备到设备、设备到云端的连接框架，允许多设备之间通信、基于云的多用户时间序列分析服务。

    • 具有 SD 卡接头，micor USB 或标准尺寸的 USB A 型插座，Micro USB 设备，6 路模拟输入，20 路输入输出引脚，一个 UART，一个 I2C，一个 ICSP 六引脚接头 (SPI)，7V-15V 直流电源输入接头。

    操作示例¶

    1. 打开网页: Grove_Indoor_Environment_Demo 下载整个项目。

    1. 点击 Tools（工具） > Serial Port（端口） 选择 Intel Edison 连接到的串口端号。

    1. 点击 Sketch（项目）>Import Library（加载库）…>Add Library 并导入**步骤一**中下载的库文件

    2. 点击 File (文件)>Examples (示例)> Grove_Indoor_Environment_Demo 选择示例并点击 upload 图标

    1. 打开 Serial Monitor（串口监视器），即可看到打印的传感器信息 :

    1. 旋转编码器检查 LCD 上显示的值。

    1. 在 “Send TextBox”，输入下面的命令对传感器及执行器进行操作 :

    set [sensor][condition:>, < or =][ threshold],[actuator]=[action]

    示例 说明
    _temp>40, relay=1_ 如果温度超过 40℃，则打开继电器
    _temp>40, sleep=1_ 如果温度超过 40℃，则不进行任何操作
    _humi>60, buzzer=1_ 如果湿度超过 60%，则打开蜂鸣器
    _light>600, servo=90_ 如果光强超过 600，舵机旋转 90°
    _uv>80, relay=0_ 如果紫外强度超过 80，关闭继电器
    _pir=1, buzzer=1_ 如果检测到人，则蜂鸣器发出警报
    _ms>40, relay=1_ 如果湿度超过 40，则继电器打开
    _ssid=name, psw=password_ 设置 WiFi 的 SSID 和密码，可打开 web 浏览器进入串口监视器或 LCD 上显示的 IP 地址，默认端口是 88，如：192.168.1.101:88

    Note

    命令应当以 ‘/n’ 结尾，所以在串口监视器中需要选中 “NewLine”。 一个执行器只能被一个传感器控制，如果 A 传感器想要控制正在被 B 传感器控制的执行器，需要先将 B 传感器设置为休眠状态。

    1. WiFi连接。打开串口监视器，设置 SSID 和密码，查看 LCD 或串口监视器上显示的本地 IP，在同一局域网中的另一个设备上打开 web 浏览器，键入上面的 IP 地址，即可看到传感器的数值。

    注意：访问 web 服务器时，需要在 IP 地址后增加端口号，如 172.20.10.2:88。
